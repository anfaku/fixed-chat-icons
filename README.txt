Wordpress Social Media Chat Plugin.

== Description ==

Visitors of your website can contact you with icons.

* WhatsApp, Messenger, Telegram, Instagram, Twitter, Mail, Call, Live Chat Icons
* Customizable Size
* Customizable Align

== Installation ==

1. Download the plugin.
2. Go to Plugins -> Add New -> Upload Plugin in your Website.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Fixed Chat Icons Plugin Settings and set your contact settings.
