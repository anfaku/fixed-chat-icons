# Fixed Chat Icons

Fixed Chat Icons Wordpress Plugin supports 7 Social Media & Contact Icon. 

 - Telegram
 - WhatsApp
 - Twitter
 - Instagram
 - Facebook Messenger
 - E-Mail
 - Quick Call
 - Live Chat URL
